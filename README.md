# HW2 — Git та структура HTML-проєкту

## Опис

Ця домашня робота виконана для практики роботи з Git та Git Bash.
У роботі було повторно використано HTML-проєкт із попереднього домашнього завдання (резюме), після чого проєкт було структуровано та оформлено як Git-репозиторій.

## У ході роботи було:

- створено структуру директорій через Git Bash;
- створено файли через консольні команди;
- додано окремі директорії для CSS, JavaScript, шрифтів та зображень;
- скопійовано зображення та шрифти через консоль;
- ініціалізовано Git-репозиторій;
- налаштовано ім’я користувача та email для Git;
- створено кілька комітів для демонстрації історії змін;
- повторно завантажено проєкт із домашнього завдання №1 у вигляді Git-репозиторію.
